Hairstyle Recommendation System

一個結合臉型分類、換臉和染髮影像處理技術的髮型推薦系統

1st Place Award Face Swapping Hair Dyeing

Project Overview

Winner of the 1st Place Award in the CYCU EL Undergraduate Project Competition, this system revolutionizes the virtual try-on experience. It provides personalized hairstyle suggestions by integrating advanced computer vision techniques for facial analysis and image synthesis.

Core Techniques

The core of the recommended system is built upon three advanced modules:

  • Advanced FA (Face Attribute Analysis): Precisely classifying face shapes to match suitable styles.
  • FS (Face Swapping): enabling users to visualize different hairstyles on their own face realistically.
  • HDT (Hair Dyeing Transfer): Simulating various hair colors to find the perfect look.
