Hairstyle Recommendation System

一個結合臉型分類、換臉和染髮影像處理技術的髮型推薦系統

1st Place Award Face Swapping Hair Dyeing

Project Overview (2022/11-2023/8)

The **Hairstyle Recommendation System** is an intelligent graphics project designed to personalize fashion experiences. This project won **1st place** in the 2023 CYCU EL undergraduate project competition and was presented at IPPR CVGIP 2023.

System Features

The system integrates multiple computer vision techniques, including **Face Shape Classification**, **Face Swapping**, and **Digital Hair Dyeing**. By analyzing the user's facial features, it suggests the most suitable hairstyles and colors, providing a realistic preview through AI-driven image manipulation.

This work demonstrates the application of deep learning in the fashion and beauty-tech industry, focusing on user interaction and visual realism.
