Dynamic Illuminance Adjustment Exposure Correction NSTC-Research Project

Project Overview (2023/10-2024/7)

This project focuses on **Dynamic Illuminance Image Enhancement** for diverse lighting conditions, ranging from extreme low-light to over-exposed scenes. It was conducted as part of an NSTC-Research Project titled "Lightweight Multi-Task Learning for Dynamic Illuminance Enhancement in ADAS".

Core Innovations

The research introduced **CPGA-Net+**, a lightweight multi-task learning architecture that incorporates **Explainable Factor Control**. By utilizing channel priors and adaptive gamma correction, the system can precisely adjust local and global illuminance without introducing artificial noise or artifacts.

Key technical focus areas include **Multi-Task Learning** to handle different exposure levels simultaneously and **Scene Analysis** to ensure safety in time-varying lighting environments.
